# Student Performance EDA using Pandas and Matplotlib

This is a beginner-level mini project based on student performance data.

In this project, I used Pandas for data handling and Matplotlib for data visualization.

## Features

- Read CSV file
- Handle missing values
- Create new column using conditions
- Groupby analysis
- Correlation analysis
- Line Plot
- Bar Plot
- Scatter Plot
- Histogram
- Save final processed CSV

## Technologies Used

- Python
- Pandas
- Matplotlib

## Files

- `main.py` → Main project file
- `students.csv` → Input dataset
- `final_output.csv` → Processed output dataset

## What I Learned

- Basic EDA workflow
- Data cleaning
- Data visualization
- Working with CSV files
- Using graphs to understand data
